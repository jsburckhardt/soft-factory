# Workitems

This directory contains all workitems. Each workitem follows the Research → Architect → Plan → Implement pipeline.

## Creating a New Workitem

1. Choose the next available ID: `WI-####-short-slug`
2. Create the directory structure:
   ```
   docs/workitems/WI-####-short-slug/
     README.md
     research/
       00-research.md
     plan/
       01-action-plan.md
       02-task-breakdown.md
       03-test-plan.md
     implementation/
       README.md
     artifacts/
       README.md
   ```
3. Start with the Research stage by filling in `research/00-research.md`
